This project is one that I created when I first started IT as a 0671 systems administrator in the Marine Corps.
Despite not doing it too often our job was to setup and configure networks and systems out in the field. I noticed
early on that we used mostly the same commands and infrastructure to set everything up. I also noticed that securecrt
had a python interpreter and decided to try my hand at learning python and automate the configuration of the network.
There was some degree of success and I tried to then take the different scripts I made to automate certian parts of
configuration and then make a centralized control center. I was then later told I should look into C# in which this 
project was born. I then made it a goal to replace securecrt and translate the previous scripts in to C#. However shortly
after I was TAD to another unit with no access to gear to test on and little time to develop anything. It now sits here
as a testiment to where I started.
